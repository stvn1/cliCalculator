calculator
==========

Expressions calculator

[![oclif](https://img.shields.io/badge/cli-oclif-brightgreen.svg)](https://oclif.io)

   
   
How to use:
- npm install (in root folder)
- npm link
- calculator "(add 1 2)"  
or    
./bin/run "(add 1 2)"


Use the calculator command to evaluate expressions
